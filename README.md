# Love Diary 💖

A modern love diary web application designed to store and preserve special memories, moments, and emotions in a beautiful and romantic interface.

## 🌸 Features
- Beautiful pink romantic UI design
- Store personal memories and moments
- Demo content included for public use
- Fully customizable structure
- Lightweight and fast web app

## 💡 About This Project
This project was originally created as a personal memory diary. All personal images and private content have been removed and replaced with demo data so that anyone can use, modify, and customize it freely.

## 🚀 Purpose
To provide a digital space where love stories and memories can be preserved in a simple and beautiful way.

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 🧠 Note
This project is built using only HTML, CSS, and a minimal amount of JavaScript. No additional frameworks or libraries have been used.
